# Creation Guide

Let's Create Your First Character!

## Quick mode or Expert mode

0️⃣ Start by choosing a mode that you prefer.

**Quick mode:** allows you to quickly build a bot with emochi helping.

**Expert mode:** can define any bot that generates more like you want,
but the generation process will ask you to make a few extra decisions
to personalize the details.
